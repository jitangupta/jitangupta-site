# Shared Context

Use this folder for project knowledge that should stay consistent across Claude, Codex, and other agents.

Recommended files:

- `brand.md` - positioning, voice, navigation, and content rules.
- `engineering.md` - stack, commands, project conventions, and implementation notes.
- `apps.md` - app/workspace structure for side projects connected to jitangupta.com.

The current canonical source remains `AGENTS.md`. These files can gradually become the smaller, reusable pieces that both `AGENTS.md` and `.claude/CLAUDE.md` point to.
