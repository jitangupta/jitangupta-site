# Product Notes

## What This Is

An AI engineering workflow guide powered by Jitan Gupta's public content and project experience.

It is not a general chatbot, lead-generation widget, or website search box.

## User Value

The user describes an engineering situation. The app recommends:

- the right workflow playbook
- practical steps
- safety checks
- prompt starter
- related Jitan videos, builds, articles, or case studies

## Quality Bar

The app should do more than retrieve content. It should help users apply workflows to real engineering situations.
